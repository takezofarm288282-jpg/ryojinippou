Googleスプレッドシート連携
📌 一度だけ設定すれば、あとは「保存」ボタンを押すだけで自動的にスプレッドシートに記録されます。
1
Googleスプレッドシートを新規作成
Googleドライブを開き、「新規 → Googleスプレッドシート」で新しいシートを作成してください。
シート名は「日報」などわかりやすい名前にしておくと便利です。
📊 スプレッドシートを新規作成
2
Apps Script を開く
スプレッドシートのメニューから
「拡張機能」→「Apps Script」 をクリック
3
以下のコードを貼り付けて保存
既存のコードを全て削除してから貼り付けてください。
※以前このアプリを設定済みの方は、列構成が変わったため（「現場ごとの実績」「残業」「残業理由」「帰宅時間」列を含む）、コードの貼り替えをおすすめします。
function doPost(e) { try { var data = JSON.parse(e.postData.contents); var sheet = SpreadsheetApp.getActiveSpreadsheet() .getActiveSheet(); // ヘッダーが無ければ追加 if (sheet.getLastRow() === 0) { sheet.appendRow([ '送信日時','日にち','現場名','住所', '現場ごとの実績', '残業時間(自動計算)','残業分析','帰宅時間','写真枚数' ]); } // データ行を追加 sheet.appendRow([ new Date().toLocaleString('ja-JP'), data.date || '', data.siteName || '', data.address || '', data.workItems || '', data.overtimeAuto || 'なし', data.overtimeReason || '', data.homeTime || '', data.photoCount || 0 ]); return ContentService .createTextOutput(JSON.stringify({result:'success'})) .setMimeType(ContentService.MimeType.JSON); } catch(err) { return ContentService .createTextOutput(JSON.stringify({result:'error', message:err.toString()})) .setMimeType(ContentService.MimeType.JSON); } }
4
デプロイ（公開）する
① Apps Script 画面右上「デプロイ」→「新しいデプロイ」
② 種類：「ウェブアプリ」 を選択
③ 実行ユーザー：「自分」
④ アクセスできるユーザー：「全員」
⑤「デプロイ」ボタンを押す
⑥ Googleアカウントの許可を求められたら「許可」を選択
⑦ 表示された 「ウェブアプリのURL」 をコピー
⚠️「アクセスできるユーザー」は必ず「全員」にしてください。
「自分のみ」にするとスマートフォンから送信できません。
5
URLをここに貼り付けて保存
✅ スプレッドシートと連携済みです。「入力」タブから日報を送信できます。
